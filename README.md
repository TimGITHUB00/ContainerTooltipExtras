# Container Tooltip Extras

Adds modified tooltips to Road to Vostok.

## Features

- RichTextLabel is now implemented into this mod to have colored text inside tooltips.
- Tooltips modified to show green colored [+] before the ContainerName when hovering over containers that have loot inside them. Yellow colored (valuables) shows after the [Open] when hovering over containers that have valuable loot inside them (like Weapons, Knives, Grenades, Electronics, Rigs, Helmets, Armor, Attachments, Ammo and Medical). (Not working inside shelters!)
- Customize which mod features are enabled via the Mod Configuration Menu (You can switch on/off: [+] and [+] (valuables)). Also change which items from these are considered as valuables: Weapons, Knives, Grenades, Electronics, Rigs, Helmets, Armor, Attachments, Ammo and Medical.

## Compatibility

- Works alongside other mods
- Requires Metro/Community Mod Loader

## Configuration

With MCM installed, find "Container Tooltip Extras" in the mod config menu:

- [+] - on/off (default: off)
- [+] (valuables) - on/off (default: off)
- Weapons as valuables - on/off (default: off)
- Knives as valuables - on/off (default: off)
- Grenades as valuables - on/off (default: off)
- Electronics as valuables - on/off (default: off)
- Rigs as valuables - on/off (default: off)
- Helmets as valuables - on/off (default: off)
- Armor as valuables - on/off (default: off)
- Attachments as valuables - on/off (default: off)
- Ammo as valuables - on/off (default: off)
- Medical as valuables - on/off (default: off)

Without MCM, settings are saved automatically with defaults so them are off!

## Installation

Drop `ContainerTooltipExtras.vmz` into your `mods/` folder and launch the game.
